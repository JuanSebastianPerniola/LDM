# LDM

Class Repository

## Description

This repository contains materials, exercises, and resources related to the **LDM** course.
It serves as a central place to organize all classwork, practical activities, and code examples, making it easier to review and track progress throughout the course.

## Contents

Some of the included folders and files:

* `Actividad1/ … ActividadN/` — One folder per activity, including the exercise statement, source code, and solutions.
* `i.html`, `index.html` — HTML files used to display or test course content.
* `README.md` — This documentation file.
* Other supporting resources (images, documents, etc.).

## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/JuanSebastianPerniola/LDM.git
   ```
2. Navigate to the folder of the activity or resource you want to review.
3. Open the HTML or source files in your preferred editor or web browser.
4. If you’re completing an exercise, copy the folder to your working environment, make the necessary changes, save your progress, and commit your updates.

## Best Practices

* Always include the activity statement along with your solution.
* Use descriptive file names (e.g., `Activity5_Solution.html`).
* Commit frequently with clear messages describing your work (e.g., “Activity 4 – added JavaScript section”).
* Before pushing, test your HTML files in the browser to ensure everything works correctly.
